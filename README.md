# Qi_OKR_2026
Analysis scripts and demo data

Matlab scripts to analyze OKR events in eye position data.

data0: eye position data (for horizontal and vertical positions)
tax: time points
tmps: templates


Add all the files in your Matlab workspace, and run "okr_detect.m" in Command Window, "okr_detect(tmps,tax,data0)".

Outputs:
Eye traces with the detected event timings.
Detected events.
Computed frequency of the events.
